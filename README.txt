                                :::ИНСТРУКЦИЯ:::

Берем из проекта только файл Game и текстовый файл TestMap

Управление:
Стрелки Вправо-влево - двигаться
X - стрелять
C - прыжок
Esc - меню




Используйте текстовый файл TestMap для взаимодействия со сценой в игре:
{
1. Укажите размерность вашей карты. Один символ - один объект на карте

2. Обозначьте символами определенные объекты:

0 - пустое место
1 = Игрок
2-4 = 3 разновидности блока, для разнообразия их внешности
5 = Ключи для выхода
6 = Враг, стреляющий  горизонтально в вас
7 = Враг Летающий вверх и вниз на один "символ"
b = летающая платформа, двигающаяся вправо на 3-3.5 "символа" и возвращающаяся обратно
с = Выход (Финиш)
d = Коробка (для создания препятствия)


Также на карте будет присутствовать:
-FallZone, касаясь которой, вы вернетесь на Респаун
-Респаун будет появляться в месте появления игрока
-Задний фон
-Время, прошедшее с начала запуска сцены

Далее, указываем информацию, соответствующую написанному, без пробела от двоеточия

Исключения:

-У Спрайтов для прыжка и бега указываем через запятую кол-во спрайтов, которые должны быть анимацией долгого использования(затянутый прыжок, разогнавшийся игрок). Если таковых нет, то ОБЯЗАТЕЛЬНО указываем 0
-У Врага-огня(стреляющего) также через запятую пишем кол-во Спрайтов для анимации стрельбы. Если таковых нет, то ОБЯЗАТЕЛЬНО указываем 0
- Есть два вида атаки:  1- Прыжок, 2-Стрельба. Если вы выбираете 1, то после запятой укажите 0, если 2, то - скорость пуль.
}

Работа со спрайтами и аудиофайлами:
{
!!!Все изображения должны быть расширения ".png" , а аудиофайлы - ".wav". Если аудиофайл не соответствует расширению, воспользуйтесь сайтом для конвертации: 
https://online-audio-converter.com/ru/

Открываем содержимое файла, далее:
Resources -> Data -> StreamingAssets

Изображение Заднего фона и финиша загружаем, не заходя в другие папки и называем соответственно "Finish.png" и "Back.png"

Что где?
Block - виды неподвижных объектов
Bullet - пуля
Enemy1 - Летающий враг
Enemy2 - Стреляющий враг
Sounds - аудиофайлы
Key - ключ
Jump - прыжок игрока
Hit - игрок, получивший урон
Stay - стоящий игрок
Run - бегущий игрок

Загружаем в соответствующие папки спрайты, называя их поочередно : "Название папки"1.png , "Название папки"2.png и т.д.
Исключение: 
-спрайты для анимации стрельбы врага называем eFire1.png и загружаем в Fire
- летающий блок - "BlockF", коробка - "Others1"

Корректируйте размер своих спрайтов, они должны быть все быть размеров 100px-200px 
Example:(Размер игрока - 120px в высоту)

В Sounds 4 вариации звуков:
-Прыжок "Jump.wav"
-Сбор ключей "Coin.wav"
-Уничтожение врага "Hit.wav"
-Задний фон "BackSound.wav"

}
